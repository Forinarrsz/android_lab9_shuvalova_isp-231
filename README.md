# Game Catalog App (Jetpack Compose)

Приложение-каталог игр, разработанное в рамках лабораторной работы по Android-разработке.

## Функционал
- Просмотр списка игр (Название, Описание, Год, Жанр).
- Переключение между играми (Вперед/Назад).
- Циклическая навигация (после последней игры идет первая).
- Переключатель "В избранное".
- Адаптивная верстка с поддержкой прокрутки.

## Использованные элементы Compose
- **Layouts:** `Column`, `Row`, `Spacer`, `Surface`.
- **Components:** `Text`, `Button`, `Switch`, `Image`.
- **Modifiers:** `fillMaxSize`, `padding`, `weight`, `verticalScroll`, `height`.
- **State:** `remember`, `mutableStateOf` для управления индексом текущей игры и статусом избранного.

## Логика работы
1. **Состояние:** В `MainScreen` хранится `currentIndex` (индекс текущей игры в списке) и `isFavorite` (булево значение).
2. **Навигация:** При нажатии кнопок индекс изменяется. Используется оператор остатка `%` для зацикливания списка.
3. **Сброс:** При переключении на другую игру флаг "Избранное" сбрасывается в false.
4. **Ориентация:** Использован `verticalScroll`, что позволяет интерфейсу корректно отображаться как в портретной, так и в ландшафтной ориентации (контент прокручивается, если не помещается).

## Структура проекта
- `MainActivity.kt`: Точка входа и основной экран.
- `UiComponents.kt`: Разбиение UI на функции (`GameImageSection`, `GameInfoSection`, `ControlPanel`).
- `GameModel.kt`: Модель данных и список игр.
